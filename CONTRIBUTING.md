# Contributing to VisionGuide AI

First off, thank you for considering contributing to VisionGuide AI! 

## Development Process
1. Fork the repo and create your branch from `main`.
2. Install dependencies via Docker or locally.
3. Run `pytest` locally to ensure your changes pass all tests.
4. Issue that pull request!

## Code Style
- We use standard Prettier formatting for frontend.
- We use Black formatting for Python code.

Thank you for helping make the world more accessible!
